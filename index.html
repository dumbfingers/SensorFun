What is Sensorfun?

Sensorfun is a project that will let you view, plot and record all the sensors that your phone is embedded with.

Currently this app is capable of recording these sensors:

  • Accelerometer
  • Gyroscope
  • Magnetic Field
  • Temperature
  • Light
  • Proximity
  • Pressure
  • Relative Humidity

Depends on what sensor your phone has shipped with, the data this app shows you may vary.

If the "Record" option is enabled, this app will write the selected sensor data separately as *.csv file to the phone's external storage, which will be convenient for you to do the analysis.

Note

  • @jfeinstein10 's Sliding Menu
